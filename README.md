# 猫健康管理アプリ

猫の健康状態を日常的に記録・管理するためのスマートフォン向けWebアプリ。

詳細仕様は [docs/spec.md](docs/spec.md) を参照。

## 開発コマンド

```bash
npm run dev      # 開発サーバー起動
npm run build    # ビルド
npm run lint     # Lint
```

## Claude Code スキル

このプロジェクトでは学習用のカスタムスキルを使えます。

### `/review` - コードレビュー

実装したコードをレビューしてもらい、理解度を深める質問をしてもらえます。

```
# ファイルを指定してレビュー
/review src/context/CatProfileContext.tsx

# 自動判定（最近変更したファイルをレビュー）
/review

# 自然言語でも呼び出せる
「レビューして」「見て」「確認して」
```

セッション終了時（「終了」「終わり」と言ったとき）に `reviews/` フォルダへ記録が自動保存されます。
