# 猫健康管理アプリ 仕様書

## 概要

1匹の猫の健康状態を日常的に記録・管理するためのスマートフォン向けWebアプリ。

- **対象ユーザー**: 個人の猫オーナー
- **対象デバイス**: スマートフォン（モバイルファースト）
- **データ保存**: localStorageのみ（サーバー不要）
- **バックアップ**: JSON形式でのエクスポート/インポート

---

## 画面構成

ボトムナビゲーションによる4タブ構成。

| タブ | パス | 概要 |
|------|------|------|
| プロフィール | `/` | 猫の基本情報の表示・編集 |
| 記録 | `/record` | 体重・食事量・体調メモの入力 |
| グラフ | `/graph` | 体重・食事量の推移グラフ |
| メモ一覧 | `/memos` | 過去の体調メモ・症状の履歴 |

---

## 機能仕様

### プロフィール画面

猫の基本情報を登録・編集する。

**登録項目:**
- 名前（テキスト）
- 生年月日（日付）
- 品種（テキスト）
- 性別（オス / メス）
- 避妊・去勢（済 / 未）
- 写真（カメラ撮影 or ライブラリから選択、base64保存）

---

### 記録画面

その日の健康データを入力する。1日1件。

**入力項目:**
- 日付（デフォルト: 今日）
- 体重（kg、小数点2桁）
- 食事量（g、整数）
- 体調メモ（自由テキスト）
- 症状タグ（複数選択可）

**症状タグ例:**
- 嘔吐、軟便、下痢、食欲不振、元気がない、飲水量増加、くしゃみ、咳

---

### グラフ画面

体重・食事量の推移を折れ線グラフで表示する。

**表示期間の切り替え:**
- 直近1週間
- 直近1ヶ月
- 全期間

---

### メモ一覧画面

過去に入力した体調メモを日付の降順で一覧表示する。

**表示内容:**
- 日付
- 体重・食事量（記録がある場合）
- 体調メモ
- 症状タグ

**操作:**
- タップで詳細表示・編集
- スワイプまたはボタンで削除

## データモデル

```ts
// 猫プロフィール（1件固定）
type CatProfile = {
  name: string
  birthdate: string        // YYYY-MM-DD
  breed: string
  sex: 'male' | 'female'
  neutered: boolean
  photoUrl: string | null  // base64 または null
}

// 日次記録（複数件）
type DailyRecord = {
  id: string               // UUID
  date: string             // YYYY-MM-DD
  weight: number | null    // kg（例: 4.23）
  foodAmount: number | null // g（例: 80）
  memo: string
  tags: string[]           // 症状タグ
}

// localStorage キー
// "cat_profile" → CatProfile (JSON)
// "daily_records" → DailyRecord[] (JSON)
```
